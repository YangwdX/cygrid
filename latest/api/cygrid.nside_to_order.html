nside_to_order¶

cygrid.nside_to_order(uint64_t nside) → uint64_t¶

Compute HEALPix order from nside.

Parameters:
nside : uint64_t

HEALPix nside parameter.
Raises:
ValueError

Invalid value for nside (must be positive).
